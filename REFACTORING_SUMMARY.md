## 🚀 Способы запуска

### Быстрый старт
```bash
# Минимальная конфигурация
TOR_PROCESSES=5 python src/new_main.py

# Полная конфигурация с валидацией
TOR_PROCESSES=15 VALIDATE_NODES=true python src/new_main.py

# Кастомный порт
LISTEN_PORT=9090 python src/new_main.py
```

### Программное использование
```python
from tor_orchestrator import TorOrchestrator

orchestrator = TorOrchestrator(listen_port=8081)
success = orchestrator.start_system(process_count=10, validate_nodes=True)

if success:
    print("Система запущена успешно!")
    status = orchestrator.get_system_status()
    print(f"Активных процессов: {status['active_processes']}")

orchestrator.stop_system()
```
